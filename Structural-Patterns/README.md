# Structural Patterns

Os Structural Patterns (Padrões Estruturais) são padrões de projeto que focam em como classes e objetos são organizados e conectados para formar estruturas maiores, mantendo o código flexível, reutilizável e de fácil manutenção.

Em vez de criar novos comportamentos, esses padrões ajudam a definir a melhor forma de compor objetos, reduzir o acoplamento entre componentes e facilitar a evolução do sistema.
